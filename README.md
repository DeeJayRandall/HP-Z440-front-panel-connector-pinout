## HP-Z440-front-panel-connector-pinout

#### Connector Overview
- Header type: Proprietary HP front‑panel connector
- Pin count: 26 pins (2 × 13)
- Function: Power button, LEDs, front I/O signaling (USB/audio are handled via separate headers/cables)
- This is the connector you must adapt if you are moving a Z440 motherboard into a standard ATX case.

#### HDD LED
- Pin 1: HDD LED +
- Pin 3: HDD LED −

#### Power LED
- Pin 2: Power LED +
- Pin 4: Power LED −

#### Power Button
- Pin 5: Power switch +
- Pin 7: Power switch −
- Shorting pins 5 and 7 momentarily will power the system on, just like a standard ATX power switch.

#### Reset Button - may not be supported
- Pin ?: Reset Button +
- Pin ?: Reset Button -

#### HP Z440 Front‑Panel Header (26‑pin)
- Board edge - Pin 6 is no-pin / key / blanked
